## Features

### 24/7 Autonomous Execution
Claude Code works while you sleep. Set a goal before bed, wake up to progress.

### Goal Decomposition
Complex goals automatically broken into 5-10 actionable tasks.

### Persistent Memory
Context remembered across sessions. AI never forgets what it was working on.

### Multi-Agent Collaboration
Multiple AI agents working in parallel with task dependency tracking.

### Smart Retry
Circuit breaker pattern handles errors automatically. No more stuck workflows.

### Token Budget
Control costs with per-day and per-minute limits.
