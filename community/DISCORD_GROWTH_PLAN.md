## Discord Server Optimization

### Channel Purpose

```
📋 General
├── #welcome          → Rules, start here
├── #announcements    → Official updates only
├── #chat            → General chat
└── #showoff         → Share projects

🤝 Support
├── #qa              → Questions & answers
├── #troubleshooting → Debug help
└── #feature-requests → Ideas

🔧 Development
├── #code-review     → PR reviews
├── #plugins         → Plugin dev
└── #internals       → Architecture

🎉 Community
├── #events          → Weekly events
├── #introductions   → New member intros
└── #random          → Off-topic
```

### Bot Setup (MEE6)

```
!setup
- Welcome channel: #welcome
- Welcome message: [embed with rules]
- Auto-role: @member (after reading rules)
- Level up messages: enabled
- XP rewards: enabled
```

### Auto-Moderation

```
Spam filter:
- Max 5 messages/10 seconds → warn
- Max 10 messages/10 seconds → mute
- Links in #qa → delete

Content filter:
- No @everyone mentions
- No political content
- No NSFW
```

## Event Hosting Guide

### Setup

1. Create event in Discord (right-click channel → Schedule Event)
2. Set reminder: 1 day before, 1 hour before
3. Prepare materials

### During Event

1. Start on time
2. Welcome everyone
3. Guide discussion
4. Capture key points
5. Thank participants

### After Event

1. Post summary in #events
2. Share resources discussed
3. Thank speakers
4. Schedule follow-up
