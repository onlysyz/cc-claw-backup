## Community Guidelines

### Code of Conduct
- Be respectful and inclusive
- Help others learn
- Give constructive feedback
- No spam or self-promotion

### Getting Help
1. Check the [docs](https://docs.cc-claw.dev)
2. Search [existing issues](https://github.com/onlysyz/cc-claw/issues)
3. Ask in [Discord #qa](https://discord.gg/cc-claw)
4. Open a new issue if needed
