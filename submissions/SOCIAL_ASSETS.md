### Short Description (for PH)
```
Make Claude Code work 24/7 while you sleep — autonomous AI coding agent with persistent memory
```

### Long Description (for PH)
```
CC-Claw transforms Claude Code into an autonomous coding partner. Set a goal, and CC-Claw
breaks it down into tasks, executes them continuously, and learns from each execution.

Features:
🎯 Goal Decomposition — Claude-powered task breakdown
⚡ 24/7 Execution — Continuous operation with smart throttling
🧠 Persistent Memory — Cross-session context retention
👥 Multi-Agent — Specialized agents working in parallel
🔄 Smart Retry — Exponential backoff + circuit breaker
🔧 12 Built-in Tools — FileProcessor, DataScraper, GitHelper, and more

Free tier available. PRO: $9.99/mo.
```
