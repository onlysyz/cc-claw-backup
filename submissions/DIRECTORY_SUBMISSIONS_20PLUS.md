## Submission Best Practices

### Before Each Submission

1. **Prepare Assets**
   - Logo (512x512 PNG)
   - Screenshots (1200x630 recommended)
   - Demo video URL (if available)

2. **Unique Descriptions**
   - Don't copy-paste between directories
   - Vary keywords based on directory focus
   - 150-300 words for full descriptions

3. **Consistent NAP**
   - Name: CC-Claw
   - URL: https://github.com/onlysyz/cc-claw
   - Description varies per directory

### Submission Order (Recommended)

**Week 1:** ProductHunt + Alternative.to (high authority, competitive)
**Week 2:** SaaSHub + AI Depot + Future Tools (developer-focused)
**Week 3:** All Things AI + Toolify + AI Finder (AI-focused)
**Week 4:** Remaining directories

### Follow-Up

- Check submission status after 7 days
- Respond to any review comments
- Update listings with new features
- Monitor for approval emails
