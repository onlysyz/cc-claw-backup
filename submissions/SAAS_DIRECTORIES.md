## SEO Best Practices

### For Each Submission

1. **Use consistent NAP**
   - Name: CC-Claw
   - Address: [your website]
   - Phone: N/A

2. **Unique descriptions**
   - Don't copy-paste — vary the wording
   - Target different keywords per directory
   - 150-300 words recommended

3. **Add images/screenshots**
   - High-quality, not watermarked
   - 1200x630 for OG images
   - White background preferred

4. **Link back strategically**
   - Homepage: canonical
   - GitHub: repository
   - Docs: relevant pages
