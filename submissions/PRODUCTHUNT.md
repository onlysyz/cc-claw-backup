## Maker Story

```
I was working on a complex API project. Every night I'd set Claude Code to help,
but when I woke up, it had stopped — rate limit hit, session expired, context lost.

I wanted something that would:
1. Work while I sleep (24/7)
2. Remember what it was doing
3. Handle errors gracefully
4. Keep making progress toward the goal

So I built CC-Claw. It wraps Claude Code in a continuous goal-driven loop
with smart throttling, persistent memory, and autonomous execution.

Now I wake up to completed tasks and progress reports.

The best part: it's all local. Your code never leaves your machine.
```
