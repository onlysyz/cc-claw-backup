## Pre-Launch Checklist

### 2 Weeks Before
- [ ] Finalize CC-Claw v0.1.0 release
- [ ] Test all features end-to-end
- [ ] Prepare ProductHunt assets (screenshots, video)
- [ ] Write ProductHunt post draft
- [ ] Notify community about upcoming launch
- [ ] Prepare Discord announcement
- [ ] Draft Twitter launch thread

### 1 Week Before
- [ ] Review all submission forms
- [ ] Prepare unique descriptions per directory
- [ ] Create/update screenshots with branding
- [ ] Set up analytics for tracking
- [ ] Prepare email list for launch notification
- [ ] Draft launch day social posts

### 3 Days Before
- [ ] Final review of all assets
- [ ] Schedule Discord reminder
- [ ] Test demo video works
- [ ] Prepare backup links/resources
- [ ] Brief any team/community helpers

### Launch Day
- [ ] Post ProductHunt at optimal time (12 AM PT Tue-Thu)
- [ ] Share Twitter thread immediately after
- [ ] Post in Discord #announcements
- [ ] Share in relevant subreddits
- [ ] Submit to priority directories
- [ ] Respond to all comments (within 2 hours)
- [ ] Thank each upvoter/commenter

### Post-Launch (Day 1-7)
- [ ] Monitor PH ranking hourly
- [ ] Update community on progress
- [ ] Follow up on all leads/questions
- [ ] Submit to remaining directories
- [ ] Collect testimonials
- [ ] Update all submissions with latest info
- [ ] Analyze traffic sources
- [ ] Plan next feature release
