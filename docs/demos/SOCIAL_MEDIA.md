## Recording Guide for Demo Videos

### Required Tools
1. **OBS Studio** (free) - Screen recording
2. **iMovie/Filmora** (or DaVinci Resolve) - Video editing
3. **Keyboard simulator** (optional) - For fake terminal input
4. **Clock app** - For time-lapse effect

### Terminal Recording Tips
```bash
# Use a dark theme for better visibility
export PS1="[CC-CLAW] \W $ "

# Record at 1080p minimum
# Terminal font: 16-18pt JetBrains Mono or similar
```

### Recommended Screen Layout
```
┌─────────────────────────────────────────┐
│  Terminal (60%)  │  Commentary (40%)    │
│  - Task output   │  - Arrow annotations │
│  - Progress      │  - Highlight key     │
│                   │    moments           │
└─────────────────────────────────────────┘
```

### Fake Terminal Animation (optional)
For demo videos, you can pre-record terminal output:

```python
# Terminal animator script
import time
import subprocess

commands = [
    "cc-claw goal '实现用户认证系统'",
    "[CC-CLAW] 🎯 Decomposing goal...",
    "[CC-CLAW] ✅ Created 9 tasks",
    "[CC-CLAW] 🚀 Starting autonomous runner...",
    # ... etc
]

for cmd in commands:
    print(cmd)
    time.sleep(1.5)
```

### Thumbnail Specs
```
YouTube: 1280x720px, <2MB
Bilibili: 1280x720px, <2MB
Twitter: 1200x675px (16:9), <5MB
```

### Recommended Flow
1. Record terminal output first (pre-recorded looks cleaner)
2. Record voiceover
3. Edit together with OBS recordings
4. Add subtitles (many users watch muted)
5. Add intro/outro animations (canva.com for templates)
