Features

Everything you need for autonomous AI work

CC-Claw wraps Claude Code in a continuous, goal-driven loop that works while you live your life.

🎯

Goal-Driven

Set a goal once. CC-Claw decomposes it into tasks and works toward completion autonomously.

⚡

Autonomous Loop

Task → Execute → Next Task → Repeat. Continuously working until the goal is reached.

🏖️

Smart Throttling

Stops on 429, checks hourly for token refresh, resumes automatically. Never wastes tokens.

🔝

Priority Queue

Your urgent instructions jump to the front of the queue. Emergency? Just send it.

🤫

Silent Mode

No periodic check-ins. Reports only when you ask, or when milestones are reached.

🔒

100% Private

All execution happens on YOUR machine. Your code. Your control. No cloud execution.

Quick Start

Get started in 5 minutes

Server Setup
# Clone the repo git clone https://github.com/onlysyz/cc-claw.git cd cc-claw # Install dependencies pip install -e . # Configure cp .env.example .env # Edit .env with your TELEGRAM_BOT_TOKEN # Start server python run_server.py
Client Setup
# Install client pip install -e . # Install CC-Claw cc-claw install # Start the daemon cc-claw start # Message the Telegram bot # Send /start to begin onboarding
