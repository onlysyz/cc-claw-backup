## What it does (500 chars)

CC-Claw is an autonomous AI working companion that turns your Claude Code plan tokens into continuous progress toward your goals.

**How it works:**
1. Message the bot (Telegram/Lark) with your goal
2. CC-Claw decomposes it into actionable tasks using AI
3. Claude Code executes tasks on YOUR local machine — private, secure
4. Smart throttling: stops on 429, checks hourly for token refresh, resumes automatically

**Built-in tools:**
- FileProcessor: read, write, search files
- DataScraper: fetch pages, extract data
- ProcessManager: list/kill processes
- GitHelper: status, diff, log, branch
- DockerHelper: ps, logs, restart

**Use cases:**
- Bug fixing on autopilot
- Feature development while you're in meetings
- CI/CD pipeline setup
- API documentation generation
